# got-char-db
# Game of Thrones Character Database

This is my WAP project.  
I made a small website where you can see characters from Game of Thrones.

## What it does (plan)
- Search character by name  
- Filter by house/family (like Stark, Lannister, Targaryen etc)  
- Sort by name or id  
- Click on card to see more details in popup  

Everything (search, filter, sort) is done only with array methods like filter(), map(), sort() — no other tricks.

## API I'm using
https://thronesapi.com/api/v2/Characters  
It's free, no key needed. Gives about 50 characters with name, house, image, title etc.

## Tech I'll use
- HTML  
- CSS
- JavaScript only  

## How to run (when it's done)
Just open index.html in browser. That's it.
